# SmartCity Citizen 🚀

A Kotlin-based Android application designed to improve urban living by enabling citizens to report issues and administrators to assign tasks to workers efficiently.

---

## 📌 Features
- **Citizen Issue Reporting**: Users can submit issues (e.g., potholes, broken streetlights, sanitation problems).
- **Admin Dashboard**: Admins can view reported issues and assign them to workers.
- **Worker Task Management**: Workers receive assigned issues and update progress.
- **Real-time Updates**: Status tracking for citizens and admins.
- **Modern UI**: Built with Jetpack Compose for a clean and responsive interface.

---

## 🛠️ Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM (Model-View-ViewModel)
- **Backend**: Firebase / REST API (configurable)
- **Database**: Room / Firestore
- **Tools**: Android Studio, GitHub Actions (CI/CD)
